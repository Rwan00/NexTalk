# NexTalk

NexTalk is a modern chat application built with **Flutter** that supports one-on-one chats and group chats. It leverages **Firebase** for backend services and **Cloud Functions** for real-time notifications. The app uses **Provider** for state management, ensuring a smooth and responsive user experience.

---

## Features

- **One-on-One Chat:** Send and receive messages in real-time with friends.
- **Group Chat:** Create and participate in group conversations.
- **Push Notifications:** Receive notifications for new messages, powered by Firebase Cloud Functions.
- **Media Sharing:** Send images and files in chats.
- **Real-Time Updates:** Changes in chat appear instantly without refreshing.
- **State Management:** Built with Provider for efficient state handling.

---

## Technologies Used

- **Flutter** - For cross-platform mobile development
- **Firebase Firestore** - Real-time database for chats and groups
- **Firebase Cloud Functions** - Sending and receiving notifications
- **Firebase Storage** - Store and retrieve media files
- **Provider** - State management for reactive UI updates
